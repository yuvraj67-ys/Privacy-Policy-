Privacy Policy

Effective Date: 28/12/2025

Hisab-Kitab (Expense Tracker) aapki privacy ko seriously leta hai. Yeh app users ke personal data ko collect ya share karne ke liye nahi banaya gaya hai. Neeche clearly explain kiya gaya hai ki app data kaise handle karta hai.

1. Data Collection

Yeh app koi personal information jaise naam, email, phone number, ya location collect nahi karta. Aap jo bhi expense data add karte ho jaise amount, category, description aur date, wo sirf aapke device ke andar store hota hai.

2. Local Storage

Saara data localStorage ke through sirf aapke mobile device par hi save hota hai. Yeh data na to kisi server par upload hota hai aur na hi kisi third-party ke saath share kiya jata hai.

3. Internet Usage

App normally bina internet ke kaam karta hai. Internet sirf third-party libraries (jaise icons, charts) ya advertisements ke liye use ho sakta hai. Aapka personal expense data internet par transmit nahi hota.

4. Advertisements

App me ads dikhaye ja sakte hain. Ads providers apni policies ke according limited non-personal data use kar sakte hain. Is app ka developer aapke expense data ko ads ke liye access nahi karta.

5. Data Security

Kyunki data sirf aapke device par store hota hai, isliye data ki security largely aapke phone ki security par depend karti hai. App kisi external ya unauthorized access ko allow nahi karta.

6. Data Deletion

Agar aap app uninstall kar dete ho, to saara stored data automatically delete ho jata hai. Developer ke paas aapka koi data save nahi hota.

7. Children’s Privacy

Yeh app sabhi age groups ke users ke liye safe hai aur jaan-bujh kar kisi child se personal information collect nahi karta.

8. Policy Updates

Privacy Policy future me update ho sakti hai. Koi bhi change isi page par update kiya jayega.
